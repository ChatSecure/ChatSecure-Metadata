ChatSecure är en gratis meddelandeapp med öppen källkod som använder OTR-kryptering över XMPP. Du kan ansluta till ett existerande Google-konto, skapa nya konton på publika XMPP-servrar (inklusive via Tor), eller till och med ansluta till din egen server för extra säkerhet.

Till skillnad från andra appar som stänger in dig i deras bubbla är ChatSecure fullt kompatibel med andra klienter som stöder OTR och XMPP, som till exempel Adium, Jitsi, med flera.

ChatSecure använder bara välkända kryptografiska bibliotek med öppen källkod för att hålla dina konversationer privata. Andra appar kan göra påståenden om "militär nivå" på säkerheten, men utan källkod som kan granskas av allmänheten och end-to-end kryptering som kan verifieras, kan du inte vara riktigt säker.

* XMPP med TLS-certifikat pinning.
* OTR för end-to-end kryptering som kan verifieras och framåtsekretess.
* OMEMO for a mobile-friendly alternative to OTR.
* Tor integration for easy access to XMPP .onion services.
* SQLCipher för att kryptera konversationsloggar lokalt.
