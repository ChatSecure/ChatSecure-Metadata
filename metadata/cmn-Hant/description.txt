ChatSecure是一套開源軟體，使用Cypherpunk的 Off-the-Record 協定為 基於XMPP (Google Talk, Jabber) 或 Oscar (AIM) 的聊天服務提供加密功能。
使用 Github 協助開發！ https://github.com/chrisballinger/ChatSecure-iOS
支援多種協定
ChatSecure 支援 XMPP、Google Talk、Facebook Chat、AIM
優秀的加密功能
Off-thre-Record 協定提供如同日常私下對話的安全特性：確定對方身份、沒人聽到。
相容性高
可和 Adium 及 Pidgin 聊天軟體互通
