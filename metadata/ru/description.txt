ChatSecure это свободное приложение для общения, с открытым исходным кодом, которое поддерживает OTR-шифрование. Вы можете использовать существующий Google-аккаунт, либо создать аккаунт на любом публичном XMPP-сервисе (в том числе через Tor), либо использовать собственный сервер для дополнительной безопасности

В отличие от программ, которые ограничивают вас собственным сервисом, ChatSecure полностью поддерживает взаимодействие с другими программами, поддерживающими OTR и XMPP, такими как Adium, Jitsi и множеством других.

ChatSecure использует только зарекомендовавшие себя себя криптографические библиотеки с открытым исходным кодом для защиты Вашей переписки. Другие приложение могут заявлять о безопасности "армейского уровня", но не предоставлять исходный код для публичной проверки.

* XMPP с привязкой TLS-сертификатов
* OTR для шифрования и прямой секретности (forward secrecy)
* OMEMO for a mobile-friendly alternative to OTR.
* Tor integration for easy access to XMPP .onion services.
* SQLCipher для локального шифрования истории переписки
* Свободное ПО: https://github.com/chatsecure/chatsecure-ios
